## List
```python
# list init
a = list() # a = []
a = [0] * 7 # a = [0, 0, 0, 0, 0, 0, 0]

# list append
a = [1, 2, 3]
a.append(4) # a = [1, 2, 3, 4]

# list pop
a = [1, 2, 3]
top = a.pop() # top = 3, a = [1, 2]

# list length
a = [1, 2, 3]
len(a) # 3

# list sort
a = [1, 2, 3]
a.sort() # 오름차순 정렬
a.sort(reverse=True) # 내림차순 정렬
b = sorted(a)
```

## If ~ in / if ~ not in
* A에는 리스트, 튜플, 문자열을 사용할 수 있다.
```python
list_ex = [1, 2, 3, 4, 5]
if 9 not in list_ex:
    print('9가 없다')
else:
    print('9가 있다')
```
